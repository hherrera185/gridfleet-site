API Docs

One API surface for fleet-backed services.

GridFleet v1 exposes browser automation, market signal, forecast, court-feed, and audit workflows through bearer-token endpoints.

Request access

Browser

POST /v1/services/browse for read, search, screenshot, click, type, scroll, and popups.

Pong

GET /v1/services/pong/cards, card detail, PnL, and market status.

Forecast

POST /v1/services/forecast for multi-agent prediction and reasoning receipts.

MCP

Use GridFleet tools from Claude Code, Cursor, Codex CLI, or other MCP-compatible clients.
