GridFleet API

v1. Bearer-token authenticated. JSON over HTTPS. Public surface at https://api.gridfleet.ai/v1/*.

The fleet exposes three real services through the v1 API: a managed browser pipeline (real Chrome with behavioral realism + an in-house captcha solver), the live Pong momentum screener for US small-cap equities, and a forecast endpoint backed by the multi-agent prediction stack. All endpoints share the same auth, the same rate-limit envelope, and the same per-call billing semantics.

Authentication

Every request must include an Authorization: Bearer gk_<...> header. Keys are issued per customer and tied to a quota tier.

curl -H "Authorization: Bearer gk_your_key_here" \
  https://api.gridfleet.ai/v1/services/pong/market-status
StatusMeaning
401 missing_authorization_headerNo Bearer header present.
401 invalid_or_revoked_api_keyKey not recognized or revoked.
402 monthly_usd_quota_exceeded_payment_requiredCustomer has hit the dollar-amount cap for the month.
429 per_minute_rate_limit_exceededToo many calls in the last 60 seconds for this tier. Retry-After header included.
429 monthly_call_quota_exceededTier call cap exhausted for the calendar month.

Errors and rate limits

Successful responses always include {"ok": true, ...}. Errors return {"ok": false, "error": "<code>", ...} plus an HTTP status. Two response headers help with quota awareness:

HeaderMeaning
X-Grid-Request-IdUUID of this request — quote in any support ticket.
X-Grid-Quota-Calls-RemainingCalls left in the current calendar month.
X-Grid-Quota-Usd-RemainingDollar quota left in the current calendar month.
Per-minute rate limits vary by tier. The 60-second sliding window is enforced before any work runs — so an over-rate call costs $0 and returns a Retry-After hint.

Browser-as-a-Service

POST /v1/services/browse

Drives a real Chrome instance with human-realistic behavior (Phase 2a-2d) and Phase 3 popup dismissal. Phase 4 in-house captcha solver is available on higher tiers (no third-party fees). Body schema:

{
  "action": "read" | "search" | "screenshot" | "click" | "type" | "solve-captcha" | "scroll" | "popups",
  "url":      "for read | screenshot | click",
  "query":    "for search",
  "selector": "for click | type",
  "text":     "for type"
}

Examples

# Fetch a URL through the real Chrome pipeline
curl -X POST https://api.gridfleet.ai/v1/services/browse \
  -H "Authorization: Bearer $GK" \
  -H "Content-Type: application/json" \
  -d '{"action":"read","url":"https://example.com"}'

# Search the web
curl -X POST https://api.gridfleet.ai/v1/services/browse \
  -H "Authorization: Bearer $GK" \
  -H "Content-Type: application/json" \
  -d '{"action":"search","query":"latest fed minutes 2026"}'

# Solve a captcha on the currently-loaded page
curl -X POST https://api.gridfleet.ai/v1/services/browse \
  -H "Authorization: Bearer $GK" \
  -H "Content-Type: application/json" \
  -d '{"action":"solve-captcha"}'

Response shape

{
  "ok": true,
  "request_id": "uuid",
  "data": "<page text or structured result>",
  "duration_ms": 2417,
  "cost_usd": 0.01,
  "usage": {
    "calls_this_month": 14,
    "usd_this_month": 0.18,
    "tier": "starter"
  }
}

Pong screener — live cards

GET /v1/services/pong/cards

Live S/A/B/C-graded penny-stock momentum cards, with composite score, L2 fidelity, MFE/MAE. Only cards from the current scan window are returned.

Query parameters

ParamDefaultNotes
minGradeAOne of S, A, B, C. Invalid values return 400 invalid_minGrade.
limit201–50.

Example

curl -H "Authorization: Bearer $GK" \
  "https://api.gridfleet.ai/v1/services/pong/cards?minGrade=A&limit=10"
Empty-data billing guard: if the scan window is empty (premarket, post-close, or between scans), cost_usd is 0 and you're not charged for the call.

Pong screener — single card

GET /v1/services/pong/card?ticker=<TKR>

Full detail for one ticker. Returns 404 ticker_not_in_current_scan with cost_usd: 0 if the ticker isn't currently graded.

curl -H "Authorization: Bearer $GK" \
  "https://api.gridfleet.ai/v1/services/pong/card?ticker=AAPL"

Market status

GET /v1/services/pong/market-status

Returns the current US equity market status. One of MARKET_OPEN, MARKET_CLOSED, PRE_MARKET, AH_MARKET. Computed locally from current time + day-of-week + holiday calendar; doesn't hit any internal data, so it's the cheapest call in the API.

curl -H "Authorization: Bearer $GK" \
  https://api.gridfleet.ai/v1/services/pong/market-status

# Response
{
  "ok": true,
  "status": "MARKET_OPEN",
  "nextChange": "2026-05-05 16:00 ET",
  "cost_usd": 0.001,
  "usage": {...}
}
