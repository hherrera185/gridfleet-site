Anatomy of HARMONIC

Eight disciplines that keep agent context honest.

HARMONIC is not a model. It is the grounding discipline that decides which context an agent is allowed to trust before it acts - decomposed here into its working parts.

H

Hierarchical

Sources are ranked. Current directives outrank stale memory, every time.

A

Agentic

Specialist roles act inside scoped lanes, not one model guessing alone.

R

Reasoning

Decisions are grounded in evidence the system can point back to.

M

Memory

Recall is bounded and provenance-tracked - useful only when its origin is known.

O

Orchestration

Work is coordinated across roles, tools, and review gates as a single flow.

N

Normalized

Raw operating state becomes sanitized, structured, comparable context.

I

Instruction

The operator's current instruction is the top of the authority stack.

C

Context

Evidence boundaries are explicit - what an agent may use, and what it may not.
