Receipts

Evidence, attached to every result.

The receipt is the product. Each response can carry the request id, the tools invoked, source references, screenshots, and verification notes - the difference between an AI that says it did the work and one that proves it.

  • Request id + timing on every call
  • Evidence stack: screenshot, source links, and verification notes
  • Clear status: queued, running, verified, needs operator review, or failed safely
  • Operator-readable notes on what was checked and what was not
